# DistilBERT GLUE Benchmarking
Run the loading_model.ipynb in order to get the files needed for benchmarking.

GLUE.ipynb
- Benchmarking code for GLUE tests for the distilBERT model 
- Libraries needed: transformers, datasets, pandas, numpy

SuperGLUE.ipynb
- Benchmarking code for SuperGlue for the distilBERT model 
- Libraries needed: transformers, datasets, pandas, numpy
